# 基于Django的论坛BBS系统 python634

## 项目概述

本项目是一个基于Python3.7和Django框架开发的论坛BBS系统，后端数据库使用MySQL5.7，前端部分使用了Vue.js。系统提供了完整的用户交互和内容管理功能。

## 技术栈

- Python 3.7
- Django
- MySQL 5.7
- Vue.js

## 功能模块

### 系统角色

- **游客**
- **用户**
- **管理员**

#### 游客

- 查看首页
- 浏览公告信息
- 阅读帖子信息
- 查看新闻资讯

#### 用户

- 登录个人中心
- 管理个人信息
- 发布和编辑帖子
- 签到积分管理

#### 管理员

- 登录后台管理
- 管理用户
- 发布和管理公告信息
- 管理帖子信息
- 管理签到积分
- 进行系统管理

### 系统功能

- **首页显示**
- **公告信息管理**
- **帖子信息管理**
- **新闻资讯**
- **个人中心**
- **用户管理**
- **签到积分管理**
- **系统管理**

## 运行环境

- Python 3.7 或更高版本
- Django 框架
- MySQL 5.7 或更高版本
- Vue.js

## 部署步骤

部署步骤将包括但不限于以下内容：
